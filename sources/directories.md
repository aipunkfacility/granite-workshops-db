```
Поиск Telegram по номеру:

```

## Workflow (3 этапа)

### Этап 1: Сбор базы

1. **JSprav** — основной источник (до 100+ компаний)
2. **2GIS** — карты, отзывы
3. **Яндекс.Карты** — карточки компаний
4. **Yell** — каталог
5. **Авито** — частные мастера

### Этап 2: Обогащение Telegram

Для каждого номера: `"{номер}" Telegram`

### Этап 3: Приоритизация

- С Telegram → приоритет для работы (`data.md`, `summary.md`, `report.md`)
- Без Telegram → остаются в `companies.md`
