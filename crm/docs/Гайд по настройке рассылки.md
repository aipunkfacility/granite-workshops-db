# **Инструкция: Настройка Email-рассылки через Gmail**

Этот гайд поможет вам запустить автоматическую рассылку по базе granite\_companies.csv, используя ваш личный аккаунт Gmail и Python.

## **1\. Подготовка аккаунта Google (Безопасность)**

Gmail блокирует вход по обычному паролю через скрипты. Вам нужен **Пароль приложения**.

1. Зайдите в [Настройки безопасности Google](https://myaccount.google.com/security).  
2. Включите **Двухэтапную аутентификацию** (если вы еще этого не сделали).  
3. В строке поиска вверху страницы введите «Пароли приложений» (App Passwords).  
4. В поле «Название» введите, например, Granite Sender.  
5. Нажмите «Создать».  
6. **Скопируйте 16-значный код.** Это ваш пароль для скрипта (вводить без пробелов).

## **2\. Подготовка базы данных**

Убедитесь, что ваш файл granite\_companies.csv находится в той же папке, что и скрипт, и имеет правильные заголовки.

Пример структуры:

Название,Email  
Гранит-Мастер,info@granit.ru  
Светлая Память,manager@memory.ru

## **3\. Установка Python и запуск**

Если у вас еще не установлен Python, скачайте его с [python.org](https://www.python.org/).

1. Создайте файл send\_emails.py.  
2. Скопируйте в него код из блока ниже.  
3. В коде укажите свой EMAIL\_SENDER и полученный 16-значный EMAIL\_PASSWORD.  
4. Откройте терминал/консоль в папке со скриптом.  
5. Запустите рассылку командой:  
   python send\_emails.py

## **4\. Рекомендации по антиспаму**

* **Лимиты:** Gmail позволяет отправлять до 500 писем в день, но для новых скриптов лучше не превышать 50-100.  
* **Паузы:** В скрипте установлена случайная пауза (20–40 сек). Не убирайте её, иначе аккаунт забанят за бота.  
* **Персонализация:** Скрипт подставляет название компании в тему и текст письма. Это снижает вероятность попадания в спам.
