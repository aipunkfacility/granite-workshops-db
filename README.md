# Granite Workshops Database

## Обзор

Сбор базы **гранитных мастерских** и **производителей памятников** по городам России.

**Целевая аудитория:** Компании, которым предлагаются услуги **ретушера фото** для обработки портретов на памятники.

> Проект переориентирован на производителей памятников (B2B) и глубокое обогащение контактных данных мессенджерами.

## Workflow (3 этапа)

### Этап 1: Сбор базы → `companies.md`

Сбор всех организаций из открытых источников:
- **Карты:** 2GIS, Яндекс.Карты, Google Maps.
- **Каталоги:** Yell, JSprav, Firmsru.
- **Инструменты:** `scripts/firecrawl_granite.py` — автоматизированный поиск через Firecrawl.

### Этап 2: Обогащение мессенджеров

Для каждого номера телефона и сайта ищется официальный Telegram, WhatsApp и VK:
- **Скрипт:** `messenger_search.py` — глубокое сканирование сайтов компаний.
- **Скрипт:** `scripts/tg_phone_finder.py` — поиск Telegram по номеру телефона.

### Этап 3: Приоритизация

- **С Telegram** → `data.md`, `summary.md`, `report.md` (приоритет для рассылки).
- **Без Telegram** → остаются в `companies.md` (в работе).

## Структура данных

```text
granite-workshops-db/
├── config.yaml              # Список 44 городов и настройки сбора
├── messenger_search.py      # Скрипт поиска мессенджеров на сайтах
├── analyze_networks_v3.py  # Скрипт поиска сетевых компаний (филиалов)
├── agents/
│   └── funeral-scraper.md  # Инструкции для AI-агентов по сбору
├── scripts/
│   ├── firecrawl_granite.py # Основной сборщик через Firecrawl API
│   ├── scrape_city.py      # Классический скрапинг городов
│   ├── scrape_fast.py      # Ускоренный сбор данных
│   ├── tg_phone_finder.py  # Поиск Telegram по номерам телефонов
│   ├── pyproject.toml      # Конфигурация зависимостей (uv)
│   └── uv.lock             # Зафиксированные версии пакетов
├── cities/
│   └── {city-name}/

│       ├── granite_companies.csv   # Итоговый CSV для CRM/Рассылки
│       ├── data.md         # Только компании с Telegram
│       └── report.md       # Отчёт с рекомендациями по городу
└── reports/
    └── progress.md         # Общий прогресс обработки городов
```

## Основные скрипты

### Сбор через Firecrawl (Рекомендовано)
Самый эффективный способ поиска именно гранитных мастерских.
```bash
python scripts/firecrawl_granite.py "Новосибирск"
```

### Поиск мессенджеров
Обогащает существующий CSV ссылками на соцсети и чаты.
```bash
python messenger_search.py
```

### Анализ сетевых компаний
Находит компании, представленные в нескольких городах.
```bash
python analyze_networks_v3.py
```

## Требования
Проект использует `uv` для управления зависимостями.
- Python 3.10+
- `firecrawl-cli` (npx)
- Playwright (для динамического скрапинга)
